# Sistema de Gestão de Clientes

Projeto simples de API REST usando Python e MySQL.

## Tecnologias
- Python (Flask)
- MySQL

## Funcionalidades
- Criar cliente
- Listar clientes
- Atualizar cliente
- Deletar cliente

## Como rodar

1. Instalar dependências:
pip install -r requirements.txt

2. Rodar o SQL no MySQL (arquivo database.sql)

3. Executar:
python app.py

Acesse:
http://localhost:5000/clientes
